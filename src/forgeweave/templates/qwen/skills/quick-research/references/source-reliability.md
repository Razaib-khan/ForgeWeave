# Quick Research: Source Reliability Scoring

When doing quick research with limited sources, score each source for reliability:

| Score | Label | Criteria |
|---|---|---|
| 5 | Official | Official documentation, specification, or maintainer source |
| 4 | Authoritative | Well-known reference (MDN, Wikipedia, npm registry) |
| 3 | Community Vetted | Stack Overflow (accepted answer), GitHub discussion with maintainer response |
| 2 | Unverified | Personal blog, forum post, AI-generated content |
| 1 | Prohibited | Blog posts, changelogs, marketing pages |

Only use sources with score 3+ for quick research answers. If you must use a score-2 source, clearly label it as "unverified" in the answer.
